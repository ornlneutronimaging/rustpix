# Installation

Rustpix can be installed in several ways depending on your needs:

| Method | Best For |
|--------|----------|
| [Python (pip)](python.md) | Python scripting, Jupyter notebooks |
| [macOS (Homebrew)](macos.md) | GUI application on macOS |
| [Rust (cargo)](rust.md) | CLI tool, Rust library development |
| [From Source](source.md) | Development, custom builds |

## Quick Install

### Python Users

```bash
pip install rustpix
```

### macOS Users (GUI)

```bash
brew tap ornlneutronimaging/rustpix
brew install --cask rustpix
```

### Rust Users (CLI)

```bash
cargo install rustpix-cli
```

## System Requirements

- **Python**: 3.11 or later
- **macOS**: Big Sur (11.0) or later, Apple Silicon (ARM64)
- **Rust**: 1.70 or later (for building from source)
